# DEMON OS — Dynamic Modular Orchestrator & Network

Автономная многопоточная система управления операционной системой Windows на базе микроядерной архитектуры, PyQt6 и локальных больших языковых моделей (LLM).

## Архитектурные особенности
* **Микроядерная архитектура**: Системное ядро (`tools.py`) содержит только критически важный низкоуровневый функционал (динамическая компиляция модулей, изолированный запуск процессов, менеджер пакетов pip).
* **Изолированная плагин-система**: Весь прикладной функционал вынесен в независимые модули внутри директории `Plugins/`. Новые инструменты компилируются в ОЗУ на лету с помощью `importlib` без перезапуска ядра.
* **Многопоточность (ThreadPoolExecutor)**: Выполнение тяжелых системных задач и генерация кода изолированы от главного GUI-потока, что полностью исключает зависание интерфейса.
* **Автономные процессы (Subprocess)**: Визуальные плагины запускаются в ОС как независимые оконные PyQt6-приложения, работающие параллельно с ядром Демона.

## Структура репозитория
* `main.py` — Графический оверлей (PyQt6), сокет-сервер локального контекста и typewriter-логика вывода.
* `tools.py` — Микроядро системы, контроллер динамического импорта и ревизии модулей.
* `brain.py` — ИИ-оркестратор, отвечающий за семантический анализ запросов и маршрутизацию вызовов функций (Function Calling).
* `Plugins/` — Директория размещения автономных пользовательских плагинов.
* `DemonShell/` — Локальное изолированное хранилище для баз данных, отчетов и сгенерированных медиафайлов.

## Спецификация окружения
* **Core Language**: Python 3.12+
* **GUI Framework**: PyQt6
* **LLM Engine**: Ollama (Qwen-2.5-7B-Instruct)
* **OS Target**: Windows 10 / 11 (с правами Администратора)
